# ETKM Skills

This folder contains all Claude skill files for East Texas Krav Maga.

## What Are Skills?
Skills are persistent instruction sets loaded into Claude at the start of relevant tasks. They encode brand voice, audience knowledge, workflow rules, and domain expertise so Claude never starts from zero.
