# Docs - ETKM Brand, Strategy, and Operational Reference

## Overview
Master documentation library for ETKM. Contains brand standards, messaging architecture,
curriculum references, and operational guides that inform all AI-generated content.

## Contents

### Brand
- Brand foundation (StoryBrand framework, controlling idea, voice)
- Brand kit (colors, typography, visual standards)
- Prohibited words and copy guidelines

### Strategy
- PEACE framework (Prepared, Empowered, Aware, Capable, Engaged)
- Audience map (14 segments with demographics and pain points)
- Problem-solution matrix per audience segment
- Story arcs for narrative content

### Curriculum
- Training program overview (Yellow through Black Belt)
- EFC law enforcement curriculum
- ACT Armed Citizen Tactics program
- Fight Back ETX womens program

### Operations
- 2026 curriculum schedule
- Student Journey Map reference
- CBLTAC event materials
- Student Etiquette Agreement

## Usage
All AI content generation must reference docs/ before producing
brand-facing copy, curriculum material, or strategic assets.
